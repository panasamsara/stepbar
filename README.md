# stepbar
jquery Plug-in stepbar (canvas)

## 参照
https://github.com/erickzanardo/stepbar    
在这位的jquery插件上做了一些小改动，一些样式的小调整    
具体使用方法，原文有写，非常简单

### 我的改动有：
     1 完成步骤的圈，变成一个颜色的实心圆，而不是两个颜色的同心圆（中间数字仍有）   
     2 线条粗细程度     
     3 步骤下面说明文字也加入了颜色配置等    
     4 当前步骤之前的，圆圈中显示对勾（√）   
     5 进度完成时,右边线条也变色
